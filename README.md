# Проект: Место

## Описание
___
"Mesto" - сервис для обмена фотографиями. 

**Функционал:**

* Просмотр фотографий с разных уголков земли
* Редактирование профиля


## Технологии, используемые в проекте:
___

* HTML5
* CSS3
* JavaScript [ES6]
* Отзывчивая верстка
* Файловая структура по методологии БЭМ (Nested)
